# Text_Analysis_project_final_2
NLP-Powered Text Insights Dashboard
 **Project Overview**
This project is an end-to-end Natural Language Processing (NLP) solution that analyzes user-generated text and converts it into meaningful insights using transformer-based models from Hugging Face.
The system processes over 1,000+ text inputs and extracts sentiment, emotions, and key phrases, presenting results through an interactive dashboard for real-time analysis.

 **Problem Statement**
Organizations receive large volumes of customer feedback in text format. Manually analyzing this data is time-consuming and inefficient.
This project automates the process by transforming raw text into actionable insights.
